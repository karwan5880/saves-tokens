# Ponytail

**Type:** minimal-implementation / “lazy senior developer” skill. **Repository:** [DietrichGebert/ponytail](https://github.com/DietrichGebert/ponytail).

## Method

It encourages the agent to identify unnecessary scope and produce the smallest sufficient implementation, reducing generated code, explanation, review, and rework.

## Cautions

This is an indirect token saver: it is valuable only when it preserves the requested behavior and required validation. It must not justify dropping security checks, accessibility, migrations, compatibility, tests, or explicit acceptance criteria.
