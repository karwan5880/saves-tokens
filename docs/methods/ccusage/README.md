# ccusage

**Type:** usage measurement, not a reducer. **Repository:** [ryoppippi/ccusage](https://github.com/ryoppippi/ccusage).

## Use

Use it to inspect/visualize Claude Code and supported agent usage records, establish baselines, and compare interventions over matched tasks.

## Caveat

Treat parsed/local records as an operational view. Provider billing and plan quota remain the source of truth where available. Always record model, reasoning level, repository revision, prompt, outcomes, and retries alongside token counts.
