# Caveman Code

**Type:** alternate terminal coding agent. **Repository:** [JuliusBrussee/caveman-code](https://github.com/JuliusBrussee/caveman-code).

## Method

It combines terse agent interaction with its own terminal-agent architecture and memory facilities, supporting many model providers.

## Evidence and cautions

Its README reports 1.93× fewer tokens than Codex in its 25-task MicroBench. This is a project benchmark: harness, model, tool policy, permissions and validation can make cross-agent comparisons non-equivalent. Validate security, provider/account compatibility, and cost per correct task before switching.
