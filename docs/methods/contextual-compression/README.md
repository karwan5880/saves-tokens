# Contextual compression

## Method

Summarize or code-aware-compress retrieved passages, tool results, and old conversation while retaining task-relevant evidence. Hierarchical summaries and compaction are forms of this technique.

## Caution

Generic compression can erase identifiers, line-level edit anchors, negations, error details, and API contracts. Retain a source reference and raw-output bypass. Judge by successful patches and provider-billed cost, not compression percentage.
