# Codex controls

## Method

Give a compact, testable task with target paths and non-goals. Keep project instructions concise, avoid large pasted context, and begin a fresh task/thread when the existing discussion is unrelated. Use the current product’s usage view and API billing where applicable.

## Note

Codex behavior and controls differ between CLI, desktop/app, cloud, and API surfaces; do not assume a setting in one exists in another. The official getting-started guidance states that prompt, high-level context and optional diff summaries are sent for generation.
