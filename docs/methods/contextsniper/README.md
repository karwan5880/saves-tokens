# ContextSniper

**Type:** code-memory / retrieval research project. **Repository:** [Calluking/ContextSniper](https://github.com/Calluking/ContextSniper).

## Method

It ranks candidate code and runtime evidence with hybrid retrieval, applies an intention-aware context gate, and returns compact evidence packets while keeping recoverable source outside the prompt.

## Evidence and cautions

Its paper reports lower token use and estimated/logged cost in pilot settings. This is promising retrieval research, not proof for every coding harness. Patch tasks can require literal source anchors, identifiers, and stack traces that a gate must preserve.
